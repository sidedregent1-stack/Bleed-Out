# HOW TO USE BLEED OUT (Complete Guide)

**Everything you need - No MelonLoader, only the C++ loader + native DLL.**

## Quick Start (Windows)
1. Build BleedOutLoader.cpp in Visual Studio 2022 (x64 Release, Multi-threaded /MT)
2. Build BleedOutNative.cpp as DLL (same settings)
3. Launch VRChat via Steam
4. Run BleedOutLoader.exe as Administrator
5. Enter path to BleedOutNative.dll when prompted
6. Press F8 in-game for the full menu
