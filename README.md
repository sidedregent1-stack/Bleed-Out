See the detailed instructions in the conversation or build the two .cpp files with Visual Studio 2022 x64 Release.

1. Build BleedOutLoader.cpp → BleedOutLoader.exe
2. Build BleedOutNative.cpp → BleedOutNative.dll
3. Launch VRChat via Steam
4. Run BleedOutLoader.exe as admin and point it to BleedOutNative.dll
